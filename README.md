# site-wukong-hub

## 项目简介

site-wukong-hub 是一个用于归档和发布多个独立 HTML 页面的仓库。  
本仓库不针对任何特定域名或网站，主要用于存储和展示结构清晰的静态页面资源，方便收藏、查阅与分发。

## 目录说明

```
.
├── pages/          # 存放各个独立 HTML 页面
├── assets/         # 页面引用的公共资源（如 CSS、JS、图片等）
├── docs/           # 相关文档或说明文件（可选）
└── README.md       # 本文件
```

- `pages/` 目录下每个文件夹或文件代表一个独立的 HTML 页面，命名应简短且有辨识度。
- `assets/` 用于存放页面依赖的静态资源，避免重复加载。
- 如有额外说明或使用文档，可放入 `docs/` 目录中。

## 页面归档说明

- 所有页面均为独立的 HTML 文件，可直接在浏览器中打开使用。
- 页面之间不互相依赖，可单独更新或删除。
- 建议在每个页面的 `<head>` 中注明版本号或更新日期，便于追踪。
- 归档内容应遵守相关法律法规，不包含违法违规或侵权信息。

## 维护说明

- 欢迎提交 Issue 或 Pull Request 来新增、修正或改进页面内容。
- 提交新页面时，请确保文件结构清晰，资源引用路径正确。
- 本仓库不承诺长期维护，但会尽力保持页面可访问与基本可用。
- 如需删除或修改已有页面，请说明理由。
